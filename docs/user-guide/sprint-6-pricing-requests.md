# Sprint 6: Pricing Requests User Guide

## Submitting a Pricing Request

1. Navigate to **Pricing Requests** in the sidebar
2. Click **+ New Request**
3. Fill in the form:
   - Select Estate, Stage, and Brand
   - Configure general toggles (Land Titled, KDRB, 10/90 Deal, etc.)
   - For Hermitage Homes: provide BDM and Wholesale Group (required)
   - Add lot entries with house type and facade type
   - Optionally add notes
4. Click **Submit Request**
5. If clash violations are detected, they will be displayed as error cards. Resolve conflicts and resubmit.

## Tracking Requests

- The list page shows all your pricing requests (or all requests for admin/pricing users)
- Filter by Status (Pending, In Progress, Completed) or Brand
- Click a request to view its detail page

## Fulfilling a Request (Admin/Pricing only)

1. Open a pending request's detail page
2. Click **Download Generated Sheet** to get the pre-filled Excel
3. Fill in pricing data in the Excel
4. Click **Upload Completed Sheet** and select the completed file
5. The system extracts packages and marks the request as Completed
6. The requester receives an in-app notification

## Resubmitting

- On any request's detail page, click **Resubmit** to create a new request pre-filled with the same form data

## Notifications

- The bell icon in the header shows unread notification count
- Click the bell to see recent notifications
- Click a notification to navigate to the related pricing request
- Use **Mark All Read** to clear the badge
- Navigate to **Notifications** page for full list with pagination
